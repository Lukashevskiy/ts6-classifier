# TS6 Feature Extractor v1.0.0

Feature extraction pipeline for T6SE training data from Bastion6.

Repository: `https://github.com/Lukashevskiy/ts6-classifier`
Dataset source: `https://bastion6.erc.monash.edu/static/download/T6SE_training_data.zip`

## Included feature groups
- AAC
- DPC
- QSO
- CTDC (7 physicochemical properties)
- CTDT (transition frequencies between groups)

## Release assets
- `REPORT.md` (submission report)
- `positive_*.csv`, `negative_*.csv`, `all_*.csv`
- `positive_*.tsv`, `negative_*.tsv`, `all_*.tsv`

## Main output files
- `positive_features.csv`, `negative_features.csv`, `all_features.csv`
- `positive_features.tsv`, `negative_features.tsv`, `all_features.tsv`
